# dominus-symfony-client
Backend for Dominus game

Ce projet a pour but de faire de la veille technologique. Il s'agit de l'interface d'administration d'un projet de jeu par navigateur, en cours de construction, et dont l'API se trouve sur le repo dominus-api.

Technos utilisées : 
* PHP8.4
* Symfony 7.2
* Typescript
* SASS
